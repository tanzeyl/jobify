## Installation

Install <strong>Jobify</strong> with npm:

````bash
git clone https://github.com/tanzeyl/jobify.git
````
Run front-end (Use this in root folder):
````bash
npm i && npm start
````
Run back-end(Use this after navigating to `backend` folder):
````bash
cd backend
nodemon ./index.js
````
